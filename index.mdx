## What dive does

dive breaks open a Docker image and lets you explore it layer by layer. You see exactly which files were added, modified, or removed at each step — and get an efficiency score showing how much space is wasted by duplicated or overwritten files.

## Get started in 30 seconds

<Steps>
  <Step title="Install dive">
    Install via your package manager of choice:

    ```bash
    # macOS
    brew install dive

    # Ubuntu/Debian
    sudo apt install dive

    # Docker (any platform)
    docker pull docker.io/wagoodman/dive
    ```
  </Step>
  <Step title="Analyze an image">
    Pass any image tag, ID, or digest:

    ```bash
    dive nginx:latest
    ```
  </Step>
  <Step title="Explore layers">
    Use `Tab` to switch between the layer list and file tree. Arrow keys navigate. `Ctrl+F` filters files.
  </Step>
  <Step title="Check CI results">
    Run in CI mode to get a pass/fail result without the interactive UI:

    ```bash
    CI=true dive nginx:latest
    ```
  </Step>
</Steps>

<Note>
  dive supports Docker, Podman, and Docker archive (`.tar`) image sources. See [Image Sources](/usage/image-sources) for details.
</Note>
